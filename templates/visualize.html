Linear Search Visualizer

Linear search is the most basic searching technique that checks each element in a list sequentially until the desired element is found or the list ends...

How to solve linear search

1. Start from the first element of the array.

2. Compare the current element with the target element.

3. If they match, return the index of the element.

4. If not, move to the next element.

5. Repeat until you find the element or reach the end.

6. If not found, return "not found".
