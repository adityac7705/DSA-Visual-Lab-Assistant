Binary Search Visualizer

Binary Search is fast (O(log n)) but requires the array to be sorted. Input will be sorted automatically before execution.

  1. Set low = 0 and high = last index.
  2. Calculate mid = (low + high) / 2.
  3. If A[mid] is target → Done.
  4. If A[mid] < target → set low = mid + 1.
  5. If A[mid] > target → set high = mid - 1.
  6. Repeat until found or low crosses high.
